# Block Search Engine Crawlers from AWS CloudFront Images

#### For web site content hosted on Amazon Web Services CloudFront, this node.js script will prevent search engine crawlers from accessing your robots.txt file.

##### Services you'll need:
* Lambda
* Cloudfront

##### Scripts you'll need:
* robots.txt.js
